# DB To UI Mapping

- `users` -> auth identity resolution and internal manager lookup.
- `clients` -> client detail, operational settings, manager ownership, targets.
- `campaigns` -> campaigns list and campaign detail.
- `leads` -> lead workspace and pipeline edits.
- `replies` -> reply history in lead detail.
- `campaign_daily_stats` -> campaign trend charts and dashboard aggregates.
- `daily_stats` -> manager/admin operational dashboard depth when populated.
